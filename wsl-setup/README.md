# WSL Setup Scripts

Scripts for configuring a development environment in Windows Subsystem for Linux (WSL).

## Contents

- `git-config.sh`: Configures Git settings (user, email, aliases, etc.).
- `nodejs-install.sh`: Installs Node.js.
- `pixi-install.sh`: Installs Pixi package manager.
- `pixi-config.sh`: Configures Pixi settings.
- `tools-install.sh`: Installs common development tools.
