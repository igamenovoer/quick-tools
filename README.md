# Quick Tools

A collection of scripts and tools to automate various daily tasks, system configurations, and development environment setups.

## Directory Structure

- **[claude/](claude/README.md)**: Scripts for setting up and configuring Claude Code and MCP tools.
- **[cuda/](cuda/README.md)**: NVIDIA CUDA repository setup.
- **[doc-processing/](doc-processing/README.md)**: Document conversion tools (HTML to Markdown).
- **[docker-ce/](docker-ce/README.md)**: Docker Engine installation on Windows (WSL2) without Docker Desktop.
- **[install-vscode-offline/](install-vscode-offline/README.md)**: Scripts for offline VS Code installation and Remote-SSH setup.
- **[linux-misc/](linux-misc/README.md)**: Miscellaneous Linux system administration scripts.
- **[others/](others/README.md)**: Uncategorized scripts.
- **[pixi/](pixi/README.md)**: Pixi package manager setup for Windows.
- **[podman/](podman/README.md)**: Podman installation and configuration for Windows.
- **[python/](python/README.md)**: Python development tools and environment management.
- **[tmux/](tmux/README.md)**: Tmux configuration scripts.
- **[via-docker/](via-docker/README.md)**: Wrapper scripts to run filesystem commands via Docker.
- **[windows-misc/](windows-misc/README.md)**: Miscellaneous Windows PowerShell scripts.
- **[wsl-setup/](wsl-setup/README.md)**: Setup scripts for Windows Subsystem for Linux (WSL).
- **[youtube/](youtube/README.md)**: YouTube video download scripts.
