# Windows 10/11 VS Code Download Scripts

PowerShell scripts to download VS Code components on Windows 10/11. These scripts are typically used on an internet-connected machine to prepare for an offline installation.

## Contents

- `get-vscode.ps1`: Downloads the VS Code installer for Windows.
- `get-code-server.ps1`: Downloads the VS Code Server binary (for the remote Linux host).
- `get-vscode-plugin.ps1`: Downloads VS Code extensions (.vsix files).
