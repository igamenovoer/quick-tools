# VS Code Offline Issues & Troubleshooting

This directory contains documentation for known issues and troubleshooting guides related to running VS Code in offline or containerized environments.

## Contents

- `issue-podman-vscode-gui.md`: Details issues related to running VS Code GUI with Podman.
- `issue-vscode-extensions-not-visible-in-terminal-container.md`: Troubleshooting guide for when VS Code extensions are not visible within a terminal container.
