# Claude Code Tools

This directory contains scripts for setting up and configuring Claude Code.

## Contents

- **Installation**:
  - `install-claude-code.sh`: Script to install Claude Code.
  - `install-nodejs.sh`: Helper to install Node.js.

- **Configuration & MCP**:
  - `add-custom-api-for-cc.{sh,ps1}`: Scripts to configure custom APIs for Claude Code.
  - `add-mcp-context7.{sh,ps1}`: Scripts to add the Context7 MCP tool.
  - `add-tavily-mcp.{sh,ps1}`: Scripts to add the Tavily MCP tool.
  - `skip-cc-login.{sh,ps1}`: Scripts to bypass Claude Code login.

- **Subdirectories**:
  - `cc-siliconflow/`: Additional resources for SiliconFlow integration.
