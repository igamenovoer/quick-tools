# Via Docker Tools

Wrapper scripts to execute standard file system commands inside a Docker container. This is useful for managing files with specific permissions or in restricted environments without installing tools on the host.

## Contents

- `ls-via-docker.sh`: List directory contents.
- `ll-via-docker.sh`: List directory contents (long format).
- `cp-via-docker.sh`: Copy files/directories.
- `mkdir-via-docker.sh`: Create directories.
- `chmod-via-docker.sh`: Change file modes/permissions.
